# Memory Center — "How Saelis remembers"

`/settings/memories` · Subtitle: _"Every memory here exists because you chose to keep it."_

Capabilities: search; filter (All / Constellations / North Stars); sort (recently updated,
recently created, alphabetical); per-memory cards showing title, full content, kind, why it may
help, created/updated dates, and whether it is in use; edit (title, content, reason, kind — with
server validation, prohibited-category and secret-material rechecks); remove (confirmed,
permanent, per the Memory Charter); clear all (confirmed, irreversible); JSON export.

Export (`saelis-memories-YYYY-MM-DD.json`) contains: export date, kind, title, content, reason,
created and updated dates — and deliberately **no** user IDs, database IDs, provider IDs, hidden
classifications, or moderation details.

Never displayed: provider confidence, hidden inference, usage counts as scores, emotional
significance, or raw internal status terminology.
